# Support Guide

## Community Support (Free)
If you have general questions or need help with configuration, please read the [GitHub Wiki](https://github.com/thisismyurl/thisismyurl-raw-support/wiki) and ask questions in the  [Discussion Area](https://github.com/thisismyurl/thisismyurl-raw-support/discussions). Currently, this is the only source for support for users who are not registered.

## Priority Support (Registered Supporters)
If you have registered the plugin to support our mission, please [open a GitHub Issue](https://github.com/thisismyurl/thisismyurl-raw-support/issues) or create an issue directly from within the plugin settings.
